# ⚽ Football Quiz App

A responsive React-based quiz application that tests your football knowledge with multiple-choice questions, instant feedback, and score tracking.

## 🚀 Live Demo

👉 https://YOUR-USERNAME.github.io/YOUR-REPO-NAME



## 📸 Features

*  Multiple-choice football quiz
*  Randomised questions every time
*  Instant feedback (correct / incorrect)
*  Score tracking
*  Progress bar
*  Final results screen
*  Restart quiz functionality
*  Responsive design



## 🛠️ Built With

* React (Create React App)
* JavaScript (ES6)
* HTML5
* CSS3

## ⚙️ Installation & Setup

Clone the repository:


git clone https://github.com/10jamescook/football-quiz.git


Navigate into the project:


cd football-quiz


Install dependencies:


npm install


Run the app:


npm start


## 🌐 Deployment

This project is deployed using GitHub Pages:


npm run deploy
